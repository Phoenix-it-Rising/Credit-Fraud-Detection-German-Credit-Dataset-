# Credit Fraud Detection using XGBoost

## Description
Implemented an XGBoost model to predict credit fraud using the German Credit dataset. Includes data preprocessing, model training, evaluation, and confusion matrix visualization.

## Tech Stack
- Python
- XGBoost
- Scikit-learn
- Pandas, Matplotlib, Seaborn

## Dataset
German Credit Data: [https://archive.ics.uci.edu/dataset/144/statlog+german+credit+data](https://archive.ics.uci.edu/dataset/144/statlog+rain/validation/test split
- Model evaluation with accuracy and classification report
- Confusion matrix visualization

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Place `german_cleaned.csv` in the project directory.
4. Run: `python german_dataset.py`
